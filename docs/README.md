# AI Helpers Website

This directory contains the GitHub Pages website for the ai-helpers project.

## How It Works

The site is generated by [skillsaw](https://github.com/stbenjam/skillsaw):

```bash
uvx skillsaw docs --format html -o docs/ .
```

## Deployment

Documentation is auto-generated by a GitHub Action after PRs merge to main.
The action opens (or updates) a PR on the `auto-update-docs` branch, which
can be merged periodically to publish updates.

GitHub Pages deploys from the `docs/` directory on `main`.

Site: https://openshift-eng.github.io/ai-helpers/
