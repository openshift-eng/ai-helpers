## Name
potato:mash

## Synopsis
```
/potato:mash
```

## Description
The `potato:mash` command mashes a potato.

## Implementation
- Print "Potato mashed!"

## Return Value
- **Format**: Success message

## Examples

1. **Basic usage**:
   ```
   /potato:mash
   ```

## Arguments
None
